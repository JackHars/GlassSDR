- [ ] Waterfall scrolls, > 20 fps, no tearing

- [ ] Tune to a different frequency; retune happens cleanly
- [ ] No RAM growth observed in Activity Monitor / Task Manager over 30 min idle in NFM mode
- [ ] Quit app via window close; backend shuts down cleanly (no zombie process)
